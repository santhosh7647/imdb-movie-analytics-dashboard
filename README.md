# 🎬 IMDb Movie Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using the IMDb Movie Dataset to analyze movie performance, ratings, revenue trends, genres, directors, and audience engagement. The dashboard transforms raw movie data into actionable insights through interactive visualizations, KPI cards, and advanced analytical reports.

The analysis covers **1,000 movies**, **644 directors**, and **996 actors** released between **2006 and 2016**.

---

## 🚀 Objectives

* Analyze movie revenue, ratings, votes, and runtime trends.
* Identify top-performing genres, directors, and movies.
* Explore relationships between ratings, runtime, and box-office revenue.
* Create an interactive dashboard for data-driven storytelling.
